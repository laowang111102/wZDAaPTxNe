# 医院后台管理系统 python295

本项目为一款基于 Python 和 Django 框架开发的医院后台管理系统，旨在为医疗机构提供全面的管理功能。系统支持二次开发，并附带完整的源代码及使用文档。

## 技术栈

- 后端框架：Django
- 数据库：SQLite
- 开发语言：Python
- 前端技术：HTML

## 功能模块

### 系统角色

- 管理员页面
  - 权限控制

### 功能页面

- 登录页面
- 管理人员/添加组页面
- 医生管理页面
- 患者管理页面
- 病房管理页面
- 科室管理页面
- 药品管理页面

### 核心操作

- 增删改查功能

## 运行环境

系统可在支持 Django 和 Python 的环境中运行。

## 部署步骤

1. 阅读使用文档，了解系统部署流程。
2. 准备运行环境，安装必要的依赖。
